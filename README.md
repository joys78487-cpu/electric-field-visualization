# Electric Field Visualization Using Numerical Simulation

This project simulates electric fields generated by point charges
using Coulomb's Law and the Superposition Principle.

## Features
- 2D grid numerical simulation
- Vector field computation
- Streamline visualization
- Dipole simulation
- PDF report generation

## Technologies
- Python
- NumPy
- Matplotlib
- ReportLab

## How to Run

```bash
python electric_field.py
